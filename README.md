- [버그 수정] : 토큰 갱신, 이미지 왜곡, 친구 검색 기능, 컴포넌트 사라짐** 등의 버그 수정했습니다.
- 슬랙 및 구글 폼을 통해 피드백을 받았고 사용자들의 의견을 바탕으로 기능을 추가했습니다. (통계 페이지)
